# Blocklists and filters for Pi-hole

This is a compilation of filters made by me, the list will get longer as time goes on.
## Regex
```
^([A-Za-z.-]*\.)?trafficjunky(.[a-z]*)
^([A-Za-z.-]*\.)?moatads\.com/
yoads(.[a-z]*)
```
## Wildcards
```
(^|\.)amazon-adsystem\.com$
(^|\.)cpmstar\.com$
(^|\.)fastfreeredics[0-9]\.life$
(^|\.)treasuredata\.com$
```
